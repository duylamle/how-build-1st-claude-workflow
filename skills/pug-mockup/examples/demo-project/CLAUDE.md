# MyApp Demo — Pug Mockup Project

> 5-page task manager demo. Open `demo.html` to preview, edit `.pug` files to customize.

## Quick Preview

Open `demo.html` in any browser — no install needed. This is the pre-built output.

## Project Structure

```
demo-project/
├── demo.html             ← pre-built output (open to preview)
├── package.json          ← build/watch scripts
├── CLAUDE.md             ← this file
├── src/                  ← SOURCE — edit files here
│   ├── index.pug         ← entry point (includes components + pages)
│   ├── components/       ← shared UI mixins
│   ├── pages/            ← 1 file per page
│   ├── css/main.css      ← design tokens + styles
│   └── js/main.js        ← SPA router
└── dist/                 ← OUTPUT — generated by `npm run build`
    └── index.html        ← compiled HTML (same as demo.html)
```

**`src/` vs `dist/`:** Edit `.pug` files in `src/`. Build compiles them into
`dist/index.html` — one self-contained HTML with CSS/JS inlined. Never edit
`dist/` directly — it gets overwritten on every build.

## Pages

| Page | File | Route | Layout |
|---|---|---|---|
| Home | `src/pages/home.pug` | `#home` | Landing — public header, hero, features |
| Dashboard | `src/pages/dashboard.pug` | `#dashboard` | Sidebar — stat cards, activity feed |
| Tasks | `src/pages/tasks.pug` | `#tasks` | Sidebar — filters, tabs, data table |
| Settings | `src/pages/settings.pug` | `#settings` | Sidebar — profile form, danger zone |
| Login | `src/pages/login.pug` | `#login` | Standalone — centered card form |

## Components

| File | Mixin | Used by |
|---|---|---|
| `src/components/header.pug` | `+headerPublic()`, `+headerLogged()` | All pages |
| `src/components/sidebar.pug` | `+sidebar('page')` | Dashboard, Tasks, Settings |
| `src/components/footer.pug` | `+footer()` | Home |

## Build

```bash
npm install          # first time only — installs pug-cli
npm run build        # compiles src/ → dist/index.html
npm run watch        # auto-rebuild on file changes
```

## How to Edit

1. Open the specific `.pug` file for the page you want to change
2. Edit the section you want
3. `npm run build` → refresh browser

Edit **components** to change something across all pages (e.g., add a sidebar link).
Edit **pages** to change one specific page (e.g., rearrange dashboard layout).
Edit **css/main.css** to change colors, fonts, or spacing globally.
